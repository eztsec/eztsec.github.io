This machine is relatively easy. In Method 1, I reviewed the usage of SQL injection. Besides extracting data, if the privileges are high enough, you can read system files, and if you know the path, you can even write a web shell. While going through the materials, I discovered that SQL injection can also lead to SSRF and RCE (link). I was asked about this during an interview before and couldn’t answer. In Method 2, pay attention to information gathering; try to be as detailed as possible, or you might miss crucial CMS information. Also, be sure to review the exploit code, or you might miss key details.
