Solution One

Host Discovery

​​

Port Scanning

​​

Access Port 8080, Discover Jenkins

​​

Vulnerability Exploitation

The version is 2.113. This version of Jenkins has an RCE (Remote Code Execution) vulnerability. Search for the exploit script.

https://github.com/orangetw/awesome-jenkins-rce-2019

​​

​​

Tip: When exploiting the vulnerability, directly appending the bash -i command after the poc will not be successful. It needs to be base64 encoded to be successfully exploited.

Successfully executed the command, and the shell was rebound.

​​

Privilege Escalation

Check the current scheduled tasks:

​​

/etc/script/CleaningScript.sh is running with root privileges every five minutes.

Check the permissions of this script:

​​

The script has permissions rwxrwxrwx, meaning any user can modify, read, and execute it. Modify the script to write the rebound shell command into it, and after five minutes, a root shell will rebound.

Successfully rebound the root shell.
